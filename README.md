## FAQ

**Does the Kill Switch block all traffic or only specific apps?**  
It blocks traffic at the network level while the VPN is down, so leaks don’t slip through.

**Will this break my local network access (printers, LAN devices)?**  
If you enable strict blocking, local traffic may be restricted. Use split rules only if you understand the tradeoff.

**How do I know the Kill Switch is actually working?**  
Connect, start a download, then force a disconnect (disable adapter / drop Wi‑Fi). Traffic should halt immediately and stay halted until the VPN reconnects.

**Is this only about the Kill Switch?**  
Kill Switch is the headline, but it also includes AES-256 encryption, No-Logs policy, Global Servers, high speed, and stable connection.
